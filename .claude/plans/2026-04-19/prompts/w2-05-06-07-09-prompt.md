### T-W2-07 — 10 샘플 unit test → **AC-2** (4h)

**경로**: `__tests__/integration/test-ac2-brainstorm-validate.sh`

**목표**: T-W2-06 `validate-output.sh`가 10개 샘플 발화에 대해 9개 이상 정확 트리거하는지 검증. **AC-2 Hard Gate**.

**10 샘플 카테고리**:
- **PASS 케이스 (4)**: 완전 답변 (advisory 없어야 함)
- **FAIL 케이스 (6)**: 각기 다른 질문 누락 (advisory 출력되어야 함)

**테스트 로직**:
```bash
#!/usr/bin/env bash
set -euo pipefail

pass=0
total=10

for fixture in __tests__/fixtures/validate-prompt/*.json; do
    expected=$(jq -r '.expected' "$fixture")  # "advisory" | "none"
    actual=$(cat "$fixture" | bash hooks/validate-output.sh 2>&1 >/dev/null && echo "none" || echo "advisory")
    if [ "$actual" == "$expected" ]; then pass=$((pass+1)); fi
done

echo "Accuracy: $pass/$total"
if [ "$pass" -ge 9 ]; then
    echo "AC-2 PASS"
    exit 0
else
    echo "AC-2 FAIL (< 9/10)"
    exit 1
fi
```

**검증**: 실행 시 정확도 9/10 이상 + 최종 출력 `AC-2 PASS`.
