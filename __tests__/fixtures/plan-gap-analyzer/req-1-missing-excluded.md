# 사용자 온보딩 플로우 개선

## Goal

신규 가입자의 7일 차 재방문율을 40% → 55%로 올린다.

## Scope

- 모바일·데스크탑 웹
- 초기 튜토리얼 3스텝

## Constraints

- 기존 인증 스택 유지
- 디자인 시스템 컴포넌트만 사용

## Success Criteria

- 7일 차 재방문율 ≥ 55%
- 튜토리얼 이탈률 ≤ 20%
